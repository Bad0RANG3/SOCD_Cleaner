# SOCD Cleaner

**WASD 键盘 SOCD 裁决工具** — 为普通键盘赋予专业 Hitbox 控制器的方向冲突处理能力。

## 这是什么

SOCD（Simultaneous Opposing Cardinal Directions，轴对向指令同时输入）指玩家同时按下相反方向键的操作，例如：

- `A + D`（同时按左和右）
- `W + S`（同时按上和下）

普通键盘会把两个键都发给游戏，导致角色卡在原地抖动。专业格斗控制器内置 SOCD 芯片来裁决这种冲突。

**SOCD Cleaner** 在系统层面拦截 W/A/S/D 按键，按你选择的策略清洗冲突，向游戏注入干净的输出——把普通键盘变成 Hitbox 级输入设备。

## 三种模式

| 模式 | 规则 | A + D 同时按时 | 适用场景 |
|------|------|---------------|---------|
| **回中** | 相反键相互抵消 | 两个键都被释放 | 格斗游戏赛事标准 |
| **后发优先** | 后按的键覆盖先按的 | 后按的键生效 | FPS 急停、CS2 |
| **先发优先** | 先按的键锁定 | 先按的键继续生效 | 横版动作游戏 |

### 持续优先（三种模式均适用）

按住 A 向左走 → 按 D 又松开 → 自动回到 A 继续向左。不需要重新按 A。

## 效果演示

打开记事本，运行 SOCD Cleaner，试试看：

```
模式：后发优先

1. 按住 A → 输出 aaaaaaaaa
2. 再按住 D → 输出 d（A 被自动释放，D 接管）
3. 松开 D   → 输出 a（D 被释放，A 自动恢复）
4. 松开 A   → 停止
```

```
模式：回中

1. 按住 A → 输出 aaaaaaaaa
2. 再按住 D → 无输出（双方抵消）
3. 松开 D   → 输出 a（A 恢复）
4. 松开 A   → 停止
```

## 快速开始

### 编译

需要 Visual Studio 2022+ 或 MSVC 构建工具：

```cmd
build.bat
```

CMake 方式：

```cmd
mkdir build && cd build
cmake ..
cmake --build .
```

### 运行

```cmd
socd.exe
```

程序静默运行在系统托盘，无窗口。右键托盘图标切换模式、退出。

## 系统要求

- Windows 10 / 11
- 普通权限即可运行（部分游戏需以管理员身份运行 SOCD Cleaner）

## 技术架构

```
物理键盘
   │
   ▼
低级键盘钩子 (WH_KEYBOARD_LL)
   │
   ├─ 非 WASD 键 → 透传给系统
   │
   └─ W/A/S/D → SOCD 裁决引擎
                  │
                  ├─ 追踪四键状态（held + 时间戳）
                  ├─ 纵轴裁决：W ↔ S
                  ├─ 横轴裁决：A ↔ D
                  ├─ 按模式输出（回中 / 后发优先 / 先发优先）
                  │
                  ▼
              SendInput 注入清洗后的按键
                  │
                  ▼
              目标应用程序
```
